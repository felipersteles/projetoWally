# Projeto Wally

O Projeto Wally busca explorar aplicações para a solução de localização Indoor utilizando beacons. O projeto consiste na implementação de um aplicativo que permite que mensagens sejam enviadas para pessoas em determinadas localizações. A localização deve ser determinada através do uso da tecnologia de beacons com implantação de modelos de aprendizado de máquina para realizar o agrupamento dos dados coletados e assim melhor classificar onde o dispositivo está localizado. Além de enviar mensagens, o sistema também deve notificar o usuário (por email e em outro dispositivo associado ao mesmo) quando um dispositivo for esquecido dentro da região mapeada pelos beacons.

Portanto, os objetivos principais do projeto são criar um sistema de envio de mensagens destinadas a uma determinada localização, implementar um mecanismo de localização indoor utilizando beacons e a notificação de dispositivos esquecidos no ambiente mapeado
